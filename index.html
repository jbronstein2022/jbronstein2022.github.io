Project Description

The crux of my project is to create a Neural Network that will use intraday stock data and news areticle concerning securities. Using reinforcement learning, the NN will decide whehther to buy, sell, or hold a position regarding a security using information from media outlets and global trends within a stock's respective subsector within the economy. The project would target large market cap stocks mainly because such stocks generate more news regarding product releases and earning reports.
